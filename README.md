Vantage AI is an intelligent document assistant designed to help students and researchers navigate complex PDFs. By leveraging the Mistral 7B LLM, Vantage AI doesn't just search for keywords; it understands the context of your study materials to provide accurate summaries, deep insights, and instant answers.

🚀 Features

Contextual PDF Chat: Ask questions about specific chapters, theorems, or data points.

Mistral 7B Integration: Powered by one of the most efficient open-source models for high-quality reasoning.

Semantic Search: Uses vector embeddings to find exactly what you need, even if the phrasing is different.

Study Summarization: Generate concise notes from lengthy academic papers or textbooks.

Privacy Focused: Designed to run locally or via secure API, keeping your documents under your control.

🛠️ Technical Stack

LLM: Mistral 7B

Language: Python

Framework: LangChain

Vector Database: FAISS / ChromaDB

Frontend: Streamlit
